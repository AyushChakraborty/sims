# Ray and bouncy ball simulation

Two simple sims, one of a ball interacting with "light" rays, and the other of a bouncy ball, to learn about C SDL.

For those who don't know, SDL (Simple DirectMedia Layer) is a cross platform development library to provide low level access to, in this case, keyboard, mouse and graphics hardware.
